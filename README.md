# 简单神经网络实现

一个用C++实现的多层感知器(MLP)神经网络，用于MNIST手写数字识别，准确率达到93%。

## 项目概述

本项目使用C++从零开始实现了一个全连接神经网络，包含完整的反向传播算法。网络支持自定义架构，并提供训练、评估和预测功能。

## 功能特性

- **纯C++实现**：不依赖任何外部机器学习库
- **可定制架构**：支持多隐藏层，可配置每层神经元数量
- **反向传播算法**：实现梯度下降和小批量更新
- **He权重初始化**：优化的权重初始化策略，提升收敛效果
- **Sigmoid激活函数**：隐藏层使用标准激活函数
- **模型持久化**：支持保存和加载训练好的模型
- **MNIST数据支持**：内置MNIST格式数据加载工具

## 技术规格

### 网络架构
- 输入层：784个神经元（28×28 MNIST图像）
- 隐藏层：2层，每层16个神经元（可配置）
- 输出层：10个神经元（数字0-9）

### 训练参数
- **训练轮数**：30次完整遍历训练集
- **学习率**：0.2
- **小批量大小**：每批50个样本
- **激活函数**：Sigmoid
- **权重初始化**：He初始化

### 数据格式
程序使用MNIST格式的CSV文件：
- 第一列：标签值（0-9）
- 后784列：归一化后的像素值（0-255）

## 构建与运行

### 环境要求
- 支持C++11的编译器
- 标准模板库(STL)

### 编译命令
```bash
g++ -std=c++11 -O2 main.cpp lib/NeturalNetwork.cpp lib/Utils.cpp -o neural_network
```

### 运行程序
```bash
./neural_network
```

## 使用方法

### 训练模式
默认配置会在提供的数据集上训练网络：

```cpp
Network net({784, 16, 16, 10});  // 输入:784, 隐藏层:16,16, 输出:10
net.train(training_data, 30, 0.2, testing_data);
```

### 预测模式
使用预训练模型进行预测：

```cpp
Network net({784, 16, 16, 10});
net.load("./result");
vector<int> predictions = net.predict(test_data);
```

### 模型保存/加载
```cpp
// 保存训练好的模型
net.save("result");

// 加载已训练模型
net.load("result");
```

## 实现细节

### 核心组件

1. **网络类** (`Net.h`)
   - 可配置层大小的网络初始化
   - 前向传播
   - 反向传播算法
   - 梯度下降优化
   - 模型序列化

2. **工具函数** (`Utils.h`)
   - CSV数据加载
   - Sigmoid激活函数
   - He权重初始化
   - 矩阵运算辅助函数

### 关键算法

- **反向传播**：实现链式法则计算梯度
- **小批量梯度下降**：每处理完一个批量后更新权重
- **交叉熵代价函数**：衡量预测误差
